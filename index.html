What is this?

Siball-Analysis, SA for short is a computer software for recording football stats... well it's no GUI... it's a boring text-based programm developed by a teenager so dont expect much.Unless you bought it :) You can find it on GitHub , it's currently open source but when the beta ends it will be paid.

What do you have now?

Right now, August 2016 there are available 3 releases in alpha and 1 in Beta. There are all stable and working. I advise to try 0.5.1 and wait for 0.5.9 to come out

Will it have buttons?Touch?

Currently only the text-based application is being developed(is the easiest one...) but there are plans on the future for a GUI application, an Android and iOS app only if there is demand

How did you do it?

The project was developed and engineered using Python with libraries like subprocess and time.
